# 👒 Monkey D. Luffy Fan Site

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)](https://www.w3.org/html/)
[![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)](https://www.w3.org/css/)

A dedicated fan-made biography website for Monkey D. Luffy, the captain of the Straw Hat Pirates from the anime *One Piece*. This project focuses on structured web design, responsive layouts, and integrating multimedia elements.

## 🌟 Key Features
- **Comprehensive Biography:** Detailed sections covering Luffy's journey, crew members, powers, and achievements[cite: 3].
- **Multimedia Integration:** Embedded audio tracks and video content to enhance the user experience[cite: 3].
- **Interactive Elements:** Includes a contact form and social media login simulation[cite: 3].
- **Custom Styling:** Clean, organized CSS structure using modern styling techniques[cite: 2].

## 🛠 Tech Stack
- **Languages:** HTML5, CSS3[cite: 3]
- **Design:** CSS Reset for consistent cross-browser styling
- **Multimedia:** Native HTML5 `<audio>` and `<video>` tags[cite: 3]

## 🚀 How to Run
1. Clone this repository to your local machine.
2. Open `index.html` directly in any modern web browser to view the site.
